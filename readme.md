Este projeto implementa um classificador de imagens usando uma rede neural convolucional (CNN) com TensorFlow e Keras. O classificador pode identificar o conteúdo das imagens em diferentes categorias (ex.: gatos, cachorros, etc.), dependendo do conjunto de dados fornecido.

Funcionalidades

Classificação Multiclasse: O modelo é configurado para classificar imagens em múltiplas categorias.
Aumento de Dados: A aplicação de aumentos de dados, como rotação e zoom, melhora a generalização do modelo.
Visualização do Treinamento: Gera gráficos de acurácia e perda ao longo das épocas para avaliar o desempenho do modelo.

O script exibirá a acurácia e a perda para cada época no terminal.
Após o treinamento, dois gráficos de acurácia e perda serão exibidos para analisar o desempenho.
